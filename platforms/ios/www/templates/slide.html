Slide effect options

  • direction: up | down | left | right
  • duration: default 400(milliseconds)
  • slowdownfactor: overlap view.
  • iosdelay: (milliseconds) wait ios view to update bofore animation start.
  • androiddelay: (milliseconds) wait Android view to update before animation start.
  • winphonedelay: forget about this...
  • fixedPixelsTop: default 0. For fixed header
  • fixedPixelsBottom: default 0. For fixed footer.
